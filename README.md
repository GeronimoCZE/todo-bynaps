# todo-bynaps

Server side REST API for a simple todo list using Express.js and PostgreSQL (pg)

## Configuration

Initialize the db
> npm run pg:init

Create the db
> npm run pg:create

Start the db
> npm run pg:start

> [!WARNING]
> You have to start the db before you start the application everytime!

Start the server (also starts the db)
> npm start
